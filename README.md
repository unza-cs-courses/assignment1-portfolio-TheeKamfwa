# Portfolio Website Assignment

This is a responsive personal portfolio website built as part of a web development assignment. The site demonstrates HTML and CSS skills, including Flexbox, CSS Grid, and the use of CSS custom properties (variables). It includes the required four sections: Home, About, Projects, and Contact, and meets accessibility and responsiveness standards.

---

## Features
- **Four main sections:** Home, About, Projects, Contact  
- **Responsive design:** Works on desktop, tablet, and mobile screen sizes  
- **CSS custom properties:** Used for consistent colors, spacing, and fonts  
- **Flexbox:** Used for at least one component (e.g., header/navigation layout)  
- **CSS Grid:** Used for at least one component (e.g., Projects section)  
- **Accessibility:** 
  - All images have descriptive alt text  
  - All form inputs have associated labels
