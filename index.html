About Me

I am a passionate and motivated web developer who enjoys creating modern, responsive, and user-friendly websites. My interest in web development started when I became curious about how websites are built and how design and code work together to create engaging user experiences. Through my studies and personal projects, I have developed strong skills in HTML5 and CSS3. I enjoy structuring web pages using semantic HTML and designing clean layouts with modern CSS techniques such as Flexbox and Grid. I also focus on responsive design to ensure that websites look great on mobile phones, tablets, and desktop devices. I believe that accessibility and usability are important aspects of web design. Creating websites that are easy to navigate and accessible to all users is a key part of my development approach. I am constantly learning new technologies and improving my skills so I can build better and more efficient web applications in the future.
